# databricksjc
Soluciones en Databricks + ingest SQL  + GITHUB Actions

Se realizo un ETL en Databricks del dataset:
  https://www.kaggle.com/datasets/sumaya23abdul/automobile-database

Contiene:
  - Preparación de Ambiente (External Location - Widgets - Grants - Vista en Power BI por conectar)
  - Datasets (.csv)
  - Proceso (Configuración - Bronze - Silver - Gold)
  - Reversión (Borrado de esquemas y tablas utilizadas)
  - Seguridad (Accesos a tablas tanto lectura y escritura)

  El ambiente trabajado en Databricks se configuro un Cluster Single Node.
  Se configuraron los Access Connectors, Storage Account (DataLake), credentials, external location.
